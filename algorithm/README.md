### 算法笔记

1. 什么是数据结构
    + Data Structure 存储数据的不同方式（数组、链表）
2. 什么是算法（Algorithm）
    + 同一问题的不同解决方法
    + 算法往往是针对特定数据结构的
3. 如何测算算法的优劣
    + 时间测算
        * 计算算法时间差
        * 幅度不够循环来凑（时间太短 多循环下）
    + 空间测算
4. Big O 时间复杂度 时间随着数据规模的扩大，时间变化的规律
    + 不考虑必须要做的操作
        * 循环、赋初值、程序初始化。。。
    + 不考虑常数项
        * 2n -- n
    + 不考虑低次项
        * n^2+n -- n^2
